How an LLM Processes a Prompt

  1. 1

    User Input

    The user types a prompt into the interface. This can be a question, command, code request, or long document.

  2. 2

    Tokenisation

    The prompt is split into smaller pieces called tokens. These tokens are the units the model can process.

  3. 3

    Embedding

    Each token is converted into a numerical representation, so the model can work with meaning and relationships mathematically.

  4. 4

    Attention Layers

    The model uses attention layers to understand which tokens are most important in relation to each other.

  5. 5

    Prediction

    The model predicts the most likely next token based on the prompt, context, and learned patterns from training.

  6. 6

    Output Generation

    The model repeats the prediction process many times until it generates a complete response for the user.
